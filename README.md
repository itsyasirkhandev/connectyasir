# Yasir Khan Portfolio

A modern, high-performance professional portfolio built with Next.js 16 and React 19.

## 🚀 Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [GSAP](https://gsap.com/)
- **Icons:** [Hugeicons](https://hugeicons.com/)
- **Components:** [shadcn/ui](https://ui.shadcn.com/)
- **Language:** TypeScript

## 🛠️ Getting Started

```bash
pnpm install
pnpm dev
```

## 📝 Development Commands

- `pnpm dev`: Start development server
- `pnpm build`: Create production build
- `pnpm check`: Run linting, type-checking, and formatting checks
- `pnpm format`: Automatically fix formatting with Prettier

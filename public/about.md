# About Yasir Khan

I build websites that help small businesses look more established online. I’m Yasir — a full-stack developer focused on modern, high-trust websites for small businesses.

## My Journey

I started building websites because I noticed a massive gap in the market. Local service businesses and small agencies often had to choose between cheap, generic templates that didn't build trust, or expensive, bloated agency retainers.

I aim to bridge that gap. I act as an autonomous web partner, delivering high-end, custom-coded web platforms with zero friction.

## Core Values

- **Complete Ownership:** I never hold your domain hostage. You own everything.
- **Transparency:** No hidden fees, no tech jargon. Just clear communication.
- **Results Over Fluff:** A beautiful site is useless if it doesn't generate business.

## Tech Stack & Capabilities

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Node.js, Next.js API Routes, Prisma
- **Databases:** PostgreSQL, Supabase
- **Hosting & CMS:** Vercel, Sanity, Stripe Integration
